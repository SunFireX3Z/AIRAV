Panduan Koneksi API

Developer dapat mengintegrasikan data telemetry unit ke dalam program lokal (seperti VSCode, Command Line, atau IoT controller) menggunakan websocket lokal.

WEBSOCKET CHANNEL INIT 
// Inisialisasi Sinkronisasi Unit
const socket = new WebSocket('ws://localhost:8080/aira/sync');

socket.onopen = () => {
  console.log('Neural sync connection active.');
  socket.send(JSON.stringify({
    action: 'CONNECT_UNIT',
    unitId: 'airi_b05'
  }));
};

socket.onmessage = (event) => {
  const telemetry = JSON.parse(event.data);
  console.log(`Mood sync: ${telemetry.affective_rate}%`);
};
